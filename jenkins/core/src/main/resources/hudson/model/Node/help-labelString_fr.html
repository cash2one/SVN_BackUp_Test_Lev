Les libellés (ou tags) sont utilisés pour grouper plusieurs esclaves dans un groupe logique.

Par exemple, si vous avez de multiples esclaves Windows et des jobs qui nécessitent Windows, taggez vos esclaves Windows avec 'windows', puis rattachez les jobs à ce tag. Cela pemettra à vos jobs de ne s'exécuter que sur les esclaves Windows et pas sur les autres.
