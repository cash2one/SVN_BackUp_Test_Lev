Метки (ярлыки) используются для группирования нескольких узлов в одну логическую группу.

Например, если у вас есть несколько узлов под управлением ОС Windows и проекты, требующие для сборки эту ОС, вы можете присвоить таким узлам метку 'windows', и затем связать проект с этой меткой. Теперь ваши сборки будут выполняться только на узлах с меткой 'windows'. Это позволяет добавлять и удалять узлы из сборочного пула, не переконфигурируя каждый из проектов, использующих эти узлы.
